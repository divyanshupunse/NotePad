# Project Name: Notepad

## Description:
Notepad is a text-based desktop application that allows users to create, view, update, and delete notes easily. Each note contains a title, category, and content, and users can manage multiple notes in a structured way. The application uses Java 8, Collections Framework, and File I/O to store notes persistently in a text file.

## Key Features:

**Add Notes**: Users can create new notes with a title, category, and content.

**View Notes**: Display all saved notes with details.

**Update Notes**: Modify content of existing notes.

**Delete Notes**: Remove notes based on the title.

**Persistent Storage**: Notes are stored in a file using FileReader/FileWriter for future access.

**Exception Handling**: Proper error handling ensures smooth execution and prevents crashes.

## Technologies Used:

- Java 8

- Collections (ArrayList)

- File I/O (FileReader, FileWriter)

- Exception Handling
